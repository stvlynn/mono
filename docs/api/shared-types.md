# API Reference: `@mono/shared` Types

## Purpose

Call out the shared types that form cross-package contracts.

## Most Important Types

- `ConversationMessage`
- `AssistantMessage`
- `ToolResultMessage`
- `UnifiedModel`
- `TaskState`
- `TaskTodoRecord`
- `TaskResult`
- `RuntimeEvent`
- `MonoGlobalConfig`
- `ResolvedMonoConfig`
- `AgentTool`

These types are the contracts most likely to require coordinated changes across packages.
